## システム要件

- Windows 10 64bit
- DirectX 12対応のGPU

## デモの操作

デモには、Mip-NeRF 360のデータを使った「Garden」と「Bicycle」のシーンが含まれています。

- 移動：WASDキー + マウス
- マップ切り替え：Tabキー
- FPS表示：Fキー
- 終了: ESCキー
