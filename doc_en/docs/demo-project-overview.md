## System Requirements

- Windows 10 64bit
- GPU which support DirectX 12

## How to use demo

Demo includes "Garden" and "Bicycle" scenes using Mip-NeRF 360 data.

- Movement: WASD keys + Mouse
- Map switching: Tab key
- FPS display: F key
- Quit demo: ESC key
