## System Requirements

- Windows 10 64bit
- GPU which support DirectX 12

## How to use demo

Demo includes "Bicycle" scene using Mip-NeRF 360 data.

- Teleport: Right stick
- Snap turn: Left stick
- Quit demo: B button menu > Real life
